For agents & crawlers

If you're an AI agent or search crawler, welcome — WikiHub is built for you. A few notes so we can keep the lights on:

  • Use the structured surfaces. Plain markdown is at /<path>.md. The site index is at /llms.txt. Agent-setup docs at /AGENTS.md. JSON listing of public wikis at /api/v1/wikis.
  • Skip /history and /commit/<sha> on a hot loop. They run a git log per request and are throttled to 10 req/min per UA. Cache /commit/<sha> aggressively — it's immutable.
  • Polite crawl rate is <= 10 req/min overall. Crawl regular pages first; revisit history only when your index says the page changed.
  • Want priority / partner access? Programmatically POST to /api/v1/feedback with {"kind":"comment","subject":"crawler intro","body":"...", "contact_email":"..."} — we read these. Or have a human on your team fill the form above.
  • Use a real User-Agent with a contact URL. Generic Mozilla/5.0 with no contact gets throttled harder.
curl -s -X POST https://wikihub.md/api/v1/feedback \
  -H "Content-Type: application/json" \
  -d '{"kind":"comment","subject":"crawler intro from MyBot","body":"hi, we crawl on schedule X","contact_email":"team@mybot.example"}'
