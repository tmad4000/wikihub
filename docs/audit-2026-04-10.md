# Spec/Code Audit — 2026-04-10

Results from parallel Claude + Codex review of spec vs implementation. Spec contradictions (#1) already fixed in this session. Remaining items are code-vs-spec gaps and missing features.

## Critical — code doesn't match spec on core behavior

- [x] **ACL precedence not enforced end-to-end.** Post-receive hook only reads frontmatter, ignores ACL globs. A file under `wiki/**` with no frontmatter gets stored as `private` in DB even if ACL says `wiki/** public`. Fix: post-receive must call `resolve_visibility()` from `acl.py`.
- [x] **Anonymous edits blocked.** Spec says `public-edit` and `unlisted-edit` allow anonymous writes. All write routes have `@api_auth_required` or `@login_required`. Need `@api_auth_optional` on write endpoints + ACL check for anonymous-write-allowed modes.
- [x] **Private-band stripping uses regex, spec mandates AST.** `git_sync.py` `strip_private_bands()` uses raw regex. Must use markdown-it's token stream to avoid stripping inside code blocks/frontmatter. Case-insensitive, unclosed bands fail closed.
- [x] **Drop `private_content` column from Page model.** Dead weight — all content lives in authoritative git repo now (spec clarified 2026-04-10). Remove column and any code paths that read/write it.

## High — core features spec'd but not implemented

- [x] **Personal wiki auto-created on signup.** Neither `auth.py` signup nor `api.py` create_account creates a wiki. Add wiki creation (slug=username) to both paths.
- [x] **`/@username` resolves to personal wiki.** Currently renders `profile.html` with wiki list. Should render personal wiki's `index.md` with other wikis listed below.
- [x] **Personal wiki can't be deleted.** No guard on `DELETE /api/v1/wikis/:owner/:slug`. Add check: if `wiki.slug == owner.username`, reject delete.
- [x] **Fork doesn't copy Page rows or reset visibility.** `fork_wiki` clones bare repo but never creates Page rows in Postgres. Forked wiki has zero pages in DB. Must index all files from cloned repo into Page table.
- [x] **Rename/move doesn't rewrite wikilinks.** PATCH endpoint moves the file but doesn't scan other pages for `[[old-path]]` references. Must scan + rewrite + commit atomically.
- [x] **Search leaks unlisted pages.** `search_pages()` includes `unlisted` and `unlisted-edit` for any authenticated user. Should only include unlisted pages owned by the current user.
- [x] **MCP server advertised but not implemented.** `.well-known` points to `/mcp` which doesn't exist. Need actual MCP server wrapping the REST API.
- [x] **Wiki delete doesn't clean up git repos.** Only deletes DB row. Must also `shutil.rmtree` both `repos/<user>/<slug>.git` and `repos/<user>/<slug>-public.git`.
- [x] **API read endpoint ignores `Accept: text/markdown`.** Always returns JSON. Must honor content negotiation.
- [x] **Git URL prefix is `/git/@user/...` not `/@user/...`** Blueprint has `url_prefix="/git"` making clone URLs wrong vs spec.
